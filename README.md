# Counter-CV
For Linux users
-----------------
* For Linux, a CMakefile is provided to compile the source code.

* * Check out the latest project source code and compile it:
```
$ git clone https://github.com/umardx/Counter-CV.git
$ cd Counter-CV
$ cd build
$ cmake ..
$ make
```
* * Run Main:
```
$ ./start_counter.sh
